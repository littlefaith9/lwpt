# Development
- make sure you have Node 22 installed in your system.
- run `npm i` to install all required dependencies.
- build the script, see "Build script".
- run `electron .` to run as development mode.

# Build script
Run as follows:
```bash
npm run ts
npm run min
```

# Build Production (Windows)
Run as follows
```bash
npm run build-exe
```
